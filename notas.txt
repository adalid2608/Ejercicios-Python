Respecto a las constantes, tenemos las siguientes:

True y False Son constantes de verdadero y falso
None Se usa para indicar que ausencia un valor

Las operaciones logicas son las siguientes:

x or y
a and y
not x

Las operaciones de comparacion son las siguientes:

x < y (x menor que y)
x > y (x mayor que y)
x <= y (x menor o igual que y)
x >= y (x mayor o igual que y)
x == y (x igual a y)
x != y (x diferente de y)
